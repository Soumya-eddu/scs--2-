# Smart Complaint System

A Flask-based web application for registering, tracking, and managing civic complaints efficiently.

## Features

* User Registration and Login
* OTP Verification
* Complaint Submission
* Complaint Tracking with Ticket ID
* Department-wise Complaint Assignment
* Citizen Dashboard
* Department Dashboard
* Status Updates
* Resolution Notes
* Proof Photo Uploads
* Complaint History
* Linked Complaints
* Auto Escalation System
* Feedback and Reopen Complaint Option
* SMS and Email Notifications
* Multilingual Support (English, Telugu, Hindi)
* Responsive User Interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Jinja Templates
* APScheduler
* Twilio API
